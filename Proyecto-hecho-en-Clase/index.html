Etiquetas para Estadísticas

Existen dos elementos que permiten llevar a cabo el control de tareas según su estado, estos son <progress> que se utiliza para definir el estadp de una tarea cuando esta aún no fue finalizada, por ejemplo el avance durante la capacitación; y <meter> que se implementa cuando necesitamos comparar los valores de una tarea cuando ya fue finalizada, por ejemplo el resultado de un examen

  • Progress

    • sin value definido es una barra que va y viene

    • Por defecto es min=0,max=1, pero eso podemos definirlo en la etiqueta, Ahora se definio value="20"; max="100"

    • value:JS
  • Meter
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum

Preprocesadores

LESS

Preprocesador que utiliza javascript como herramienta de compilacion para crear hojas de estilo. Su principal ventaja es la asincronicidad del lenguaje que permite al usuario llamar a los estilos sin importar el orden de aparicion en las lineas de codigo, para compilarlo debemos utilizar el comando lessc input.less output.css

Contactanos

Ingrese su nombre solamente con letras y espacios. Evite catacteres que no se encuentren en el inglés. El mismo debera contar con al menos 3 caracteres y un maximo de 50 caracteres.
Ingrese una dirección de correo valida que respete el siguiente formato "usuario@servidor.com". La misma debe contar con al menos 8 caracteres.
Ingrese su número telefónico como figura en el ejemplo: (5411)2345-6789. Puede insertar un guión cada 3 a 6 dígitos.
